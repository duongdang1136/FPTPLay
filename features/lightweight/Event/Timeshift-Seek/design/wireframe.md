│                                             │
│─────────────────────────────────────────────│

## Platform 4: Box/OTT (Android TV / Fire OS)

> Very similar to SmartTV. Key difference: physical remote with dedicated rewind/fast-forward buttons on some models.

### State A: At Live Edge (Overlay visible)

```
╔══════════════════════════════════════════════════════════════════════╗
║                                                                      ║
║                    ← Player Video Area →                             ║
║                                                                      ║
║  ┌─────────────────────────────────────────────────────────────────┐ ║
║  │ Tên Sự Kiện                                    ⬤ TRỰC TIẾP    │ ║
║  │─────────────────────────────────────────────────────────────────│ ║
║  │ ◄──────────────────────────────────────────────────────────────●│ ║
║  │ 19:00                                                     20:45  │ ║
║  │                                                                  │ ║
║  │  [⏪ -30s]  [⏮ -10s]  [▶/⏸]  [⏭ +10s]  [⬤ LIVE]  [Menu]    │ ║
║  └─────────────────────────────────────────────────────────────────┘ ║
╚══════════════════════════════════════════════════════════════════════╝
```

### Notes

- Box remote: D-pad left = −10s, right = +10s; hold = −30s/+30s per interval.
- "⬤ LIVE" button navigates to live edge when focused + OK pressed.
- LIVE badge is red when at edge; badge disappears (replaced by jump-to-live) when behind.
