# Researcher Input Contract

## Required inputs

- `state/tasks/{TASK-ID}/state.yaml`
- Original user intent and task type.

## Optional inputs

- Existing Wiki/RAG hits.
- Project source files and docs.
- External research when local knowledge is insufficient.

## Priority

1. Existing project/code evidence.
2. Workspace Wiki knowledge.
3. External sources.
4. Stated assumptions when evidence is missing.
